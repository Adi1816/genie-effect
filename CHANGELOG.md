## 0.1.0 - 2026-05-04

Initial public release.

### Added

- `runGenieEffect` vanilla DOM API
- `useGenieEffect` React hook
- Required runtime stylesheet export
- TypeScript definitions
- Support for element, `DOMRect`, and plain rectangle targets
- Auto direction detection for upward or downward minimize motion
- Source hiding and restoration during animation
- Cancellation API with `onCancel`
- `prefers-reduced-motion` support
- React demo app
- MIT license
